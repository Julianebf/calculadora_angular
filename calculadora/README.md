# 🧮 Calculadora Angular

Projeto simples de uma calculadora desenvolvida com Angular.

## 🚀 Funcionalidades

* Soma de dois números
* Interface simples e responsiva

## 🛠️ Tecnologias utilizadas

* Angular
* TypeScript
* HTML
* scss

## 📦 Como executar o projeto

```bash
# instalar dependências
npm install

# rodar o projeto
ng serve
```

Acesse no navegador:
http://localhost:4200/

## 📁 Estrutura

* `minhacalculadora.component.ts` → lógica da aplicação
* `minhacalculadora.component.html` → interface
* `minhacalculadora.component.scss` → estilos
